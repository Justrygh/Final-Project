## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
```sh
python3
python3-pip
python3-dev
postgresql
postgresql-client
dnsutils
net-tools
autoconf
automake
build-essential
libtool
default-jdk
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/Justrygh/Final-Project.git
```
2. Install the following packages
```sh
libgetdns10
libgetdns-dev
libcurl4-openssl-dev
libssl-dev
libev4
libev-dev
libevent-2.1-7
libevent-core-2.1-7
libevent-openssl-2.1-7
libevent-dev
libuv1
```
