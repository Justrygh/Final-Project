Welcome!
Before we begin please take a moment to read the instructions below.
This script is built for LINUX operation system only!
In this script we are going to change our DNS resolver!

First locate your Index.py file and open the terminal in the directory its located.
Make sure your user is root, if not please enter the following line in the terminal:
       sudo su
(You will be asked to provide your user password)

Finally, run the script with the following command:
       python3 Index.py
